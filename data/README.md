# About This Data #

## Mine Data ##
The clean data includes the following meta data about domestic aggregate extraction:
`Mine ID`,
`Mine Name`,
`Mine Status`,
`Type of Mine`,
`Street`,
`City`,
`State`,
`Zip Code`,
`Latitude`,
`Longitude`

- Spatial Resolution: state-level
- Temporal Resolution: 2025
- Source: https://www.msha.gov/data-and-reports/mine-data-retrieval-system
